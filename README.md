# Customer Complaint Management System

A MERN Stack based application for managing customer complaints efficiently.

## Features

- User Registration and Login
- JWT Authentication
- Complaint Creation
- Complaint Tracking
- Complaint Assignment
- Status Updates
- Role-Based Access Control

## Technologies Used

- MongoDB
- Express.js
- React.js
- Node.js

## Project Modules

### Customer
- Register and Login
- Create Complaints
- Track Complaint Status

### Admin
- Manage Users
- View Complaints
- Assign Complaints to Agents

### Agent
- View Assigned Complaints
- Update Complaint Status
